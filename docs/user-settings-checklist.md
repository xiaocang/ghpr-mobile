# User Settings Checklist (Push Ready)

Use this list in onboarding and diagnostics screens.

## Required

- GitHub account linked
- Target repositories selected
- Android notification permission granted

## Recommended

- Notification categories (review requested / mention / comment / CI failed)
- Quiet hours
- Open-refresh minimum interval
- Push-triggered refresh enabled

## Diagnostics items

- FCM token registered
- GitHub App installation confirmed
- Last webhook activity timestamp
- Last successful push dispatch timestamp
